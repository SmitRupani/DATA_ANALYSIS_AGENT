# Data Analysis Agent

## Overview

This project implements a data analysis agent that processes datasets (CSV, JSON, Excel) and performs analysis using an LLM-based workflow.

---

## Features

* Data loading and schema extraction
* Supports CSV, JSON, and Excel files
* Execution engine for analysis
* LLM integration for intelligent responses

---

## Setup Instructions

### 1. Clone the repository

```
git clone <your-repo-url>
cd DATA_ANALYSIS_AGENT
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

---

## Running the Project

```
python app.py
```

---

## Notes

* Make sure required datasets are available
* Update file paths if needed
* Requires Python 3.x
